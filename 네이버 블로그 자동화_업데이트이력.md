# 네이버 블로그 자동화 — 업데이트 이력

## v1.1.3 (2026-07-08) — 현재 버전

- Windows 버전에서 "즉시 발행" 클릭 시 버튼 배치가 흐트러지며 2줄로 밀리던 문제 수정 (macOS는 영향 없음)

## v1.1.2 (2026-07-08)

- 썸네일에 들어가는 문구를 제목과 별도로 생성하는 기능 추가 (핵심 키워드 포함 18~20자, 마음에 안 들면 직접 수정 가능)
- 썸네일 문구가 목표 글자수를 벗어나지 않도록 강제하는 안전장치 추가
- AI가 "모든 것" 같은 상투적인 표현을 덜 쓰도록 제목·썸네일 문구·도입부·본문·마무리 전체에 규칙 강화
- 실제로는 정상 발행됐는데도 앱에는 "발행중"으로 영구히 멈춰 표시되던 버그 수정
- 이미지 자동 첨부 화면 레이아웃 개선 (부연설명 표시 방식, 키워드 입력칸 크기)
- 다른 컴퓨터로 옮겨 실행할 때 발생하는 호환성 문제 대응용 빌드 도구 추가

## v1.1.1 (2026-07-07)

- 완전자동 발행 시 썸네일·이미지·본문·관련 사이트 중 하나라도 빠지면 발행을 막고 검수 대기로 돌리는 안전장치 추가
- 검수 대기 중인 글을 "글 생성" 화면으로 옮겨 직접 수정한 뒤 발행할 수 있는 기능 추가
- 본문 서식 스타일 5종(색상·아이콘) 추가 — 랜덤 적용 또는 직접 선택 가능
- 소제목만 있고 내용 없이 바로 다음 제목으로 넘어가던 문제 수정
- 발행 전 미리보기 기능 추가 (수동·반자동 발행 시 실제 발행 모습을 미리 확인)
- 제목 공식, 키워드 밀도, 마무리 CTA 등 SEO 최적화 프롬프트 규칙 추가
- 본문에 들어가는 사진을 3장에서 5장으로 확대
- 썸네일 배경 사진을 직접 선택(수동) 또는 자동 랜덤 선택(반자동·완전자동)하는 기능 추가
- 본문 대·중·소제목 구조를 자동으로 검증하고 보정하는 기능 추가
- 이미지 검색이 한글 키워드로 실패하면 영어로 재검색하도록 개선

## v1.1.0 (2026-07-06)

- 새로 추가한 계정에서 네이버 카테고리 로드가 타임아웃되던 문제 수정
- 자동화 루프에서 계정별로 실제 네이버 블로그 카테고리를 지정해 발행하는 기능 추가 (등록된 실제 카테고리명 자동 조회)
- 계정 1개에 카테고리 배정을 여러 개(최대 5개)까지 등록해 돌아가며 발행하는 기능 추가
- 글감 수집 입력칸에 지우기(×) 버튼 추가, 카테고리 값이 키워드 등록 후에도 유지되도록 개선
- 자동화 루프 발행 시 언스플래시 사진이 삽입되지 않던 문제 수정
- 자동화 루프의 계정 간 발행 대기 시간 로직 수정 (등록 계정 전체가 한 바퀴 돈 뒤에만 대기하도록)
- 키워드 분석·트렌드 등록으로 추가한 키워드에 카테고리가 저장되지 않던 버그 수정

## v1.0.9 (2026-07-05)

- 완전자동·반자동 자동화 루프 기능 추가 — 글감 수집부터 글 생성, 발행까지 자동으로 이어서 진행 (대시보드에서 모드 선택 후 시작/중지)
- 반자동 모드에서 자동 생성된 초안을 확인 후 발행하는 "검수 대기" 화면 신규 추가
- 완전자동 발행 안전 설정이 권장 범위(하루 최대 발행 횟수·발행 간격)를 벗어나면 경고 문구 표시
- 자동화 루프 계정별로 서로 다른 글감 카테고리를 배정해 계정 간 콘텐츠 중복을 막는 기능 추가
- 키워드 삭제 시 관련 수집 데이터가 함께 삭제되지 않던 문제 수정
- 글감을 정상적으로 수집했는데도 "신규 0개"로 잘못 표시되던 오류 수정
- 화면 전반의 레이아웃과 여백을 대시보드 기준으로 통일

## v1.0.8 (2026-07-04)

- 제목 재생성 시 소제목·불릿 기호가 섞여 나오던 오류 수정 (이제 항상 한 줄 평문)
- 도입부/마무리 재생성 시 본문에서만 써야 할 소제목 구조가 섞여 나오던 오류 수정
- 썸네일 제목이 길 때 글자가 겹치던 문제 수정 (길이에 맞춰 자동으로 글자 크기 축소)
- 새 버전이 나오면 앱 실행 시 자동으로 알려주고 업데이트하는 기능 추가

## v1.0.7 (2026-07-03)

- 즉시 발행 시 일일 발행 한도를 무시하고 무제한 발행되던 오류 수정

## v1.0.6 (2026-07-02)

- 썸네일 디자인 전면 개편 (포토 프레임형 — 사진 배경, 테두리, 코너 뱃지)
