- 계정 추가 시 뜨는 네이버 로그인 창에 "로그인 상태 유지" 체크박스를 빨간 테두리와 안내 문구로 강조 표시 — 체크해두면 로그인 세션이 더 오래 유지되는 데 도움이 될 수 있어 놓치지 않도록 안내

  - 본문 안에 최소 한 곳은 "단계별 절차"나 "항목별 비교"를 구조화된 목록으로 담도록 해서, 막연한 설명 대신 바로 참고할 수 있는 정보가 들어가도록 함
  - "경험담 포함" 옵션을 켰는데도 실제로는 1인칭 경험 문장이 하나도 안 들어가는 경우가 있던 것을 자동으로 확인해, 빠져 있으면 한 번 더 보완하도록 개선
  - "글감 수집"으로 미리 모아둔 실제 관련 블로그 글 정보를 글 생성 시 참고 자료로 활용하도록 개선 — 확인 안 된 구체적인 가격·업체명을 함부로 단정하지 않고, 실제 존재하는 정보를 참고해 더 근거 있는 내용을 쓰도록 유도(다만 완전한 실시간 사실 검증은 아니며, 향후 별도로 검토할 예정)
- 자동 생성된 키워드나 본문에 "2026 식자재마트 할인"처럼 연도가 어색하게 억지로 붙어 문장이 부자연스러워지던 문제 개선 — 키워드 자동 생성 시 연도가 붙은 키워드를 최소화하고, 제목·썸네일 문구·본문 어디에도 연도를 기계적으로 반복 삽입하지 않도록 수정
- "글 톤"을 감성형으로 설정해도 정보형처럼 딱딱하게 나오던 문제 수정 — 톤마다 실제로 문체가 달라지도록 구체적인 작성 지침 보강(예: 감성형은 개인적 소감·감탄 표현을 적극 사용하도록), 다른 규칙과 충돌해 문체가 딱딱해지던 부분도 함께 조정
- 본문에 자동 삽입되는 사진(썸네일 제외 5장)이 글 초반에 몰리던 문제 수정 — 사진이 들어가는 위치를 글 전체에 고르게 분산되도록 개선
- 글 생성 화면에 "관련 사이트"를 직접 추가·수정·삭제할 수 있는 편집 기능 추가 — 지금까지는 AI가 자동으로 채운 내용만 그대로 쓸 수 있었는데, 이제 이미지 자동 첨부 영역 아래에서 사이트 이름과 주소를 직접 입력하거나 필요없는 항목을 선택해서 지울 수 있음

## v1.1.7 (2026-07-17)

- **완전자동 발행 시 실제로는 실패했는데 "발행 완료"로 잘못 표시되던 문제 수정** — 원인은 이전 세션에서 작성 중이던 글이 있을 때 네이버 에디터가 띄우는 "작성 중인 글이 있습니다" 안내 팝업이 실제 편집 화면을 가리고 있던 것으로 확인됨(수동 발행 때는 사람이 직접 팝업을 확인하고 취소를 눌러왔으나, 완전자동 모드는 이 팝업의 존재 자체를 인지하지 못해 그 뒤로도 붙여넣기·클릭을 계속 "시도"만 하고 실제로는 아무 것도 반영되지 않았음). 이제 팝업을 자동으로 감지해 취소 처리한 뒤 정상적으로 새 글 작성을 진행함.
- **발행 카테고리 이름이 네이버 자체 내장 기능("내돈내산 상품 첨부")과 우연히 같은 경우, 엉뚱하게 그 기능 버튼을 클릭해 관련 없는 동의창이 뜨고 이후 발행 패널 전체가 막히던 문제 수정** — 카테고리를 찾을 때 이 내장 기능 문구가 포함된 후보는 제외하도록 개선.
- 완전자동 루프 실행 중 특정 조건에서 발행이 되지 않고 오류로 중단되던 문제 수정 (내부 설정값 참조 오류)
- 네이버 에디터 하단에 "전체 글감" 패널이 추가된 뒤로, 발행 과정에서 태그가 엉뚱한 곳(글감 검색창 등)에 잘못 입력되던 문제 수정 — 발행 전 태그 입력 로직을 정비하고, 발행 패널이 항상 정상적으로 펼쳐지도록 이 패널을 자동으로 접어두는 처리 추가
